# O que é UML?

* UML - Unified Modeling Language (Linuagem de Modelagem Unificada)
  
É uma linguagem de modelagem, que permite visualizar de forma padronizada o projeto (design) de um sistema

# Historia 
Desenvolvida por **Grady Booch**, **Ivar Jacobson** e **james Rumbaugh** quando travalhavam na Rational Software entre 1994 e 1995.
* O Object management grup (OMG) adotou a UML como linguagem padrão em 1997, e em 2005 foi publicado pela ISO como um padrão aprovado.
